# dq_integration Module

## Purpose
- Connects the platform to external systems such as cloud storage, Power Platform tools, and notification channels.

## Sub-packages
- `azure_blob/`: upload retrieval, storage management, and event hooks for Azure Blob Storage.
- `power_platform/`: connectors for Power Apps, Power BI exports, and Microsoft Flow/Power Automate.
- `notifications/`: adapters for email, webhooks, and collaborative tools (e.g., Microsoft Teams).

## When to use
- Building integrations with customer ecosystems.
- Coordinating with IT teams on connectivity, credentials, and message formats.
- Extending notifications or exports to new channels.
